# Github

## Github 터미널 계정 변경

```bash
git config user.name
git config user.email
git config --global user.name ngenebio
git config --global user.email <user email>
```

```bash
git clone git@github.com:squidfunk/mkdocs-material-insiders.git mkdocs-material
```
